DOMES-Acoustics 用户指南

v1.2.0 2026-05-11
🌊

专业水声数据分析平台

DOMES-Acoustics 是一款面向海洋工程与水声学研究的专业软件，提供从信号采集、频谱分析、目标识别到物理仿真的全流程解决方案。

核心功能模块

📊

频谱视图

实时 LOFAR 显示，支持 WAV 音频文件的时频分析

🎯

声源定位

波束形成与 DOA 估计，支持多种阵列配置

🔬

物理仿真

声场传播损失计算，射线/简正波/抛物近似模型

🤖

AI 助手

深度学习目标识别，31 类海洋哺乳动物分类

🗺️

平台部署

无人平台轨迹可视化与时空数据分析

🏷️

标注管理

时频标注与 JSON 导出，支持批量处理

系统要求

操作系统 Windows 10/11, macOS 12+, Linux (Ubuntu 20.04+)
浏览器 Chrome 110+, Firefox 110+, Edge 110+, Safari 16+
内存 最低 4GB，推荐 8GB 以上
存储 最低 500MB 可用空间
音频 支持 Web Audio API 的声卡

快速入门

1

启动软件

打开浏览器，访问 DOMES-Acoustics 主页面。系统会自动检测浏览器兼容性。

2

导入数据

点击左侧导航栏的"频谱视图"，通过"Upload WAV"按钮导入音频文件。支持 .wav 格式，采样率不限。

3

分析信号

使用播放控制条播放音频，LOFAR 图会实时滚动显示。鼠标悬停可查看频率和时间信息。

4

导出结果

在标注管理器中框选感兴趣区域，导出 JSON 格式的标注文件，用于后续分析。

💡
提示：首次使用时建议导入一个短音频文件（< 30秒）熟悉界面操作。大文件（> 100MB）可能需要较长的处理时间。

频谱视图 (Spectrum View)

功能说明

频谱视图提供实时的 Low-Frequency Analysis and Recording (LOFAR) 显示，用于观察音频信号的时频特征。

操作指南

操作 说明
Upload WAV 导入 .wav 格式的音频文件，支持单声道/立体声
Play / Pause 播放或暂停音频，LOFAR 图同步滚动
进度条拖动 跳转到指定时间点，音频和图像同步更新
鼠标悬停 显示当前位置的频率值（Hz/kHz）和时间戳

技术参数

  • FFT 大小：2048 点
  • 频率上限：8 kHz（可配置）
  • 颜色映射：蓝-青-绿-黄-红（经典声学配色）
  • 滚动速度：2 像素/帧
  • 平滑系数：0（无平滑，原始响应）
⚠️
注意：大文件解码时会显示"Processing audio..."状态。解码完成后才能播放。如果解码失败，请检查文件是否为标准 WAV 格式。

声源定位 (Source Localization)

功能说明

基于传感器阵列的波束形成技术，估计水下声源的到达方向 (DOA)。支持多种阵列配置和算法。

阵列类型

类型 描述 适用场景
HLA 水平线列阵 (Horizontal Linear Array) 拖曳阵、海底阵
VLA 垂直线列阵 (Vertical Linear Array) 深海探测、潜标
Circular 圆环阵 (Circular Array) 全方位覆盖、浮标

DOA 算法

Bartlett

常规波束形成器，计算简单，鲁棒性好，分辨率较低。

经典

MVDR (Capon)

最小方差无失真响应，自适应抑制干扰，分辨率中等。

自适应

MUSIC

多重信号分类，超分辨率算法，可分辨多个信号源。

高分辨

配置参数

  • 阵元数量：4-256 个，典型值 16
  • 阵元间距：0.001-10 m，典型值 0.05 m
  • 声速：1000-1600 m/s，典型值 1500 m/s
  • 频率带：1 Hz - 采样率/2
  • 积分时间：0.1-10 s
  • 波束数量：10-720，典型值 360

物理仿真 (Physical Simulation)

功能说明

计算水下声场的传播损失 (Transmission Loss, TL)，模拟声波在海洋环境中的传播过程。

传播模型

模型 原理 适用条件
射线模型 几何声学近似，追踪声线路径 高频 (> 500 Hz)，距离远
简正波 波动方程本征函数展开 低频 (< 500 Hz)，水平不变
抛物近似 抛物方程步进求解 复杂环境，水平变化

环境参数

  • 声源深度：0 - 海底深度
  • 声源频率：1 Hz - 100 kHz
  • 声源级：50 - 230 dB re 1μPa
  • 最大距离：1 - 1000 km
  • 海底深度：10 - 11000 m
  • 深度网格：0.1 - 10 m

海底类型

淤泥 (Silt) 声速 1500 m/s，密度 1.4 g/cm³
沙 (Sand) 声速 1650 m/s，密度 1.6 g/cm³
粘土 (Clay) 声速 1550 m/s，密度 1.5 g/cm³

可视化输出

  • 传播损失热力图：距离-深度二维分布
  • 声速剖面图：深度方向的声速变化
  • TL 曲线图：固定深度的传播损失随距离变化

AI 助手 (AI Assistant)

功能说明

基于 SeaFusionNet-V 深度学习模型，对水声信号进行自动目标识别与分类。

识别类别

鲸类 (Cetaceans)

  • 抹香鲸 (Sperm Whale)
  • 座头鲸 (Humpback Whale)
  • 小须鲸 (Minke Whale)
  • 蓝鲸/长须鲸 (Fin Whale)
  • 露脊鲸 (Right Whale)
  • 弓头鲸 (Bowhead Whale)
  • 白鲸 (Beluga)
  • 独角鲸 (Narwhal)
  • 虎鲸 (Killer Whale)
  • 伪虎鲸 (False Killer Whale)

海豚类 (Dolphins)

  • 宽吻海豚 (Bottlenose Dolphin)
  • 普通海豚 (Common Dolphin)
  • 飞旋海豚 (Spinner Dolphin)
  • 斑海豚 (Spotted Dolphin)
  • 条纹海豚 (Striped Dolphin)
  • 瑞氏海豚 (Risso's Dolphin)
  • 领航鲸 (Pilot Whale)
  • 瓜头鲸 (Melon-headed Whale)
  • 弗氏海豚 (Fraser's Dolphin)
  • 糙齿海豚 (Rough-toothed Dolphin)

海豹类 (Pinnipeds)

  • 海象 (Walrus)
  • 髯海豹 (Bearded Seal)
  • 竖琴海豹 (Harp Seal)
  • 豹海豹 (Leopard Seal)
  • 罗斯海豹 (Ross Seal)

使用流程

1

导入音频

点击"导入音频数据"，选择多个 WAV 文件（支持批量上传）

2

选择文件

勾选需要识别的文件，点击"开始批量识别"

3

查看结果

识别完成后显示目标类别和置信度，支持导出报告

💡
建议：为获得最佳识别效果，建议使用信噪比 > 10 dB、时长 > 5 秒的音频片段。模型对 50-500 Hz 频段的鲸类叫声识别率最高。

平台部署 (Platform Deployment)

功能说明

可视化海洋无人平台（浮标、潜标、AUV、USV 等）的空间轨迹与部署位置。

支持的数据格式

格式 扩展名 说明
Excel .xlsx, .xls Microsoft Excel 格式，支持多 Sheet
CSV .csv 逗号分隔文本，通用格式

必需字段

  • 平台 ID：buoy_id / platform_id / id
  • 经度：buoy_lon / longitude / lon（单位：度）
  • 纬度：buoy_lat / latitude / lat（单位：度）

CSV 示例

buoy_id,buoy_lon,buoy_lat,timestamp
A1,121.4737,31.2304,2026-01-15T08:00:00Z
A1,121.4837,31.2404,2026-01-15T09:00:00Z
A1,121.4937,31.2504,2026-01-15T10:00:00Z
B2,121.5037,31.2204,2026-01-15T08:00:00Z
B2,121.5137,31.2104,2026-01-15T09:00:00Z

交互操作

操作 说明
鼠标滚轮 缩放 X 轴（横向）
Shift + 滚轮 缩放 Y 轴（纵向）
滑块拖动 独立控制 X/Y 轴缩放范围
Reset View 恢复初始视图范围
Snapshot 导出 PNG 格式图像（2x 分辨率）

环境数据 (Environment Data)

功能说明

浏览和管理已知的水声目标特征库，包括各类船舶、海洋生物和环境噪声的标准特征。

数据库内容

🚢

船舶噪声

  • 商船 (Merchant Vessels)
  • 渔船 (Fishing Boats)
  • 游艇 (Yachts)
🐋

海洋生物

  • 齿鲸 (Toothed Whales)
  • 须鲸 (Baleen Whales)
  • 海豚 (Dolphins)
  • 海豹 (Seals)
  • 海狮 (Sea Lions)
🌊

环境噪声

  • 雨噪声 (Rain Noise)
  • 风成噪声 (Wind Noise)
  • 热噪声 (Thermal Noise)
  • 冰噪声 (Ice Noise)
  • 生物 chorus (Biological Chorus)
🔧

人工声源

  • 气枪 (Air Guns)
  • 声纳 (Sonars)
  • 水下施工 (Underwater Construction)
  • 地震勘探 (Seismic Survey)

特征参数

每个目标条目包含以下特征信息：

  • 频谱特征：主频、谐波结构、带宽
  • 时域特征：脉冲重复周期、持续时间
  • 空间特征：传播损失模型、指向性
  • 季节分布：出现概率随月份变化
  • 地理分布：典型出现海域

标注管理 (Annotation Manager)

功能说明

对水声信号的时频图进行交互式标注，支持框选区域并导出结构化标注数据。

标注流程

1

导入音频

点击"IMPORT WAV"导入音频文件，支持批量导入多个文件

2

选择文件

在左侧文件列表中点击要标注的文件，等待 LOFAR 图渲染完成

3

框选标注

在频谱图上按住鼠标左键拖动，框选感兴趣的区域

4

修改类别

在表格中选择标注的类别（Vessel/Whale/Mechanical/Ambient）

5

导出数据

点击"SAVE & DOWNLOAD"导出 JSON 格式的标注文件

标注属性

属性 说明 单位
时间起点 标注区域开始时间 秒 (s)
时间终点 标注区域结束时间 秒 (s)
频率下限 标注区域最低频率 赫兹 (Hz)
频率上限 标注区域最高频率 赫兹 (Hz)
持续时间 时间终点 - 时间起点 秒 (s)
类别标签 目标类型分类 -

导出格式 (JSON)

[
  {
    "category": "Vessel",
    "timeStart": 12.50,
    "timeEnd": 18.30,
    "freqLow": 150,
    "freqHigh": 800,
    "duration": 5.80
  },
  {
    "category": "Whale",
    "timeStart": 25.00,
    "timeEnd": 35.00,
    "freqLow": 50,
    "freqHigh": 500,
    "duration": 10.00
  }
]
⚠️
注意：标注操作依赖精确的像素坐标映射。如果窗口大小发生变化，已创建的标注框位置可能出现偏差。建议在完成所有标注后再调整窗口大小。

数据格式规范

音频文件要求

格式 WAV (RIFF/WAVE)
编码 PCM 16-bit 或 24-bit
声道 单声道 (Mono) 或立体声 (Stereo)
采样率 任意（推荐 ≥ 16 kHz）
最大时长 无限制（大文件处理时间较长）

轨迹数据要求

平台部署模块支持以下字段名（自动识别）：

// 平台标识
buoy_id, platform_id, id, device_id

// 经度（度）
buoy_lon, longitude, lon, lng, x

// 纬度（度）
buoy_lat, latitude, lat, y

// 时间戳（可选）
timestamp, time, datetime, date

标注数据格式

标注管理器导出的 JSON 文件包含以下字段：

{
  "category": "Vessel",      // 类别标签
  "timeStart": 12.50,      // 开始时间 (s)
  "timeEnd": 18.30,        // 结束时间 (s)
  "freqLow": 150,          // 最低频率 (Hz)
  "freqHigh": 800,         // 最高频率 (Hz)
  "duration": 5.80         // 持续时间 (s)
}
